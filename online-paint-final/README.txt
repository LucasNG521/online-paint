Online Paint Web App by Stephen, Lucas, and Parker

FEATURES/////////////////
- Paint feature with variable line width and color change
- Create basic shapes with variable line width, fill colors, and advanced pattern options.
- Create your own shape with the polygon feature, also with variable line width and color fills
- Includes a color picker pallete and an eyedropper for precise color usage
- Undo and redo your drawings
- Canvas size is adjustable up to a width of 2000 x 1000px, with an added feature to clear the canvas on resize!

TECHNOLOGY USED/////////////
- This web app was written using javascript, along with bootstrap plugins and jquery
